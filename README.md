# Invoice Generator (Python)

A simple and professional Python-based invoice generator that creates PDF invoices with auto-incrementing invoice numbers.

## Features
- Generate PDF invoices
- Auto-incrementing invoice numbers
- Save invoice data locally
- Custom client and item details
- Simple CLI interface

src/ - core application logic  
data/ - stored invoice data  
output/ - generated invoices  

## How It Works
1. User enters client and invoice details
2. Invoice data is saved locally
3. A PDF invoice is generated and stored
